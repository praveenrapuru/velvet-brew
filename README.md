# Velvet Brew Café

A café website built with React and Tailwind CSS.

## Features
- Dark/Light mode toggle
- Menu with category filters (coffee, desserts, snacks)
- Table booking form with validation
- Animated stats counter
- Responsive design

## Tech Stack
- React
- Tailwind CSS
- React Router

## Setup

1. Clone the repo
```
git clone https://github.com/yourusername/velvet-brew-cafe.git
cd velvet-brew-cafe
```

2. Install dependencies
```
npm install
```

3. Add images to `public/images/` folder:
- hero-bg.jpg
- about.jpg
- cappuccino.jpg, latte.jpg, coldbrew.jpg, espresso.jpg, mocha.jpg
- tiramisu.jpg, cheesecake.jpg, brownie.jpg, croissant.jpg
- avocado-toast.jpg, sandwich.jpg

4. Run the project
```
npm start
```

Opens at http://localhost:3000
